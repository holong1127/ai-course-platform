## 本課要點

- MCP 係 AI 嘅「萬能插座」
- Vision MCP 可以等任何 AI 都識睇圖
- Claude Code 同 OpenCode 都適用

---

## 1. MCP 回顧

MCP（Model Context Protocol）係 AI 嘅「USB-C 接口」——統一咗 AI 工具嘅擴展標準。

之前嘅 Vision MCP 可以令任何支援 MCP 嘅 AI 工具都識睇圖。

---

## 2. 安裝 Vision MCP

> 幫我安裝 Vision MCP

AI 會問你幾個問題：
1. 你用咩 AI 工具？（OpenCode / Claude Code）
2. 你用咩系統？（Windows / Mac）
3. 配置放喺邊？

答完之後，AI 會自動幫你裝好。

---

## 3. 三種睇圖方式總結

| 方式 | 優點 | 缺點 |
|---|---|---|
| 多模態模型 | 直接用，唔使配置 | 模型較貴 |
| Observer 子代理 | 平價模型 + 識圖能力 | 需要 oh-my-opencode-slim |
| Vision MCP | 任何 AI 都用得 | 需要配置一次 |

---

## 關鍵收穫

Vision MCP 係最通用嘅方案，裝一次就任何 AI 工具都用得。
