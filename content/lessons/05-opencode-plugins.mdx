## 📌 1. 點解要裝插件？

默認嘅 OpenCode 就好似一個**啱啱返工嘅實習生**——你叫佢做咩佢就做咩，做得好唔好全靠彩數。

裝咗插件之後，佢就變成咗一個**成熟嘅會計團隊**：

| 冇插件 | 有插件 |
|---|---|
| 你講咩佢做咩 | 先同你確認需求，再規劃，再執行 |
| 一個 AI 做晒所有嘢 | 唔同任務分俾擅長嘅人做 |
| 直接上手寫 Code | 先出方案、拆任務、再動手 |

接下來要裝嘅兩個插件，一個負責「想清楚先做」，一個負責「搵啱人開工」。

## 📌 4. 避坑指南

### Plan / Build 模式消失咗？

呢個係最常見嘅問題。解決方法：

1. 去 OpenCode **設定 → 通用 → 高級**
2. 打開「自定義智能體」開關
3. 如果仲係唔得，打開 `opencode.jsonc`，將 `plan.disable` 同 `build.disable` 改為 `false`

### 模型列表變空？

設定文件俾插件覆蓋咗。刪除 `.config/opencode` 資料夾重新設定，或者叫 AI 幫你修復。
