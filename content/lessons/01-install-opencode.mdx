## ⬇️ 2. 下載同安裝

<div className="h-4"></div>

### Step 1：打開官網

喺瀏覽器搜尋 **OpenCode**，入官網後按 **Download**。

<a href="https://opencode.ai/" target="_blank" rel="noopener noreferrer">opencode 官網</a>

下載頁面提供三個版本：

| 版本 | 說明 |
|---|---|
| **桌面版（Desktop）** | 推薦新手使用，圖形界面，double click安裝 |
| **Terminal 版（Terminal）** | 喺Terminal中運行，適合進階用戶 |
| **擴展版** | VS Code 等編輯器插件 |

<div className="h-4"></div>

### Step 2：下載安裝包

根據你嘅系統選擇：
- **Windows**：下載 `.exe` 安裝包
- **Mac**：下載 `.dmg` 安裝包

<div className="h-4"></div>

### Step 3：安裝

double click安裝包，按提示完成安裝。

> 安裝過程中唔需要任何設定，一路下一步就得。

## 📂 3. 打開項目資料夾

安裝完成後，打開 OpenCode，會見到一個選擇項目資料夾嘅界面。

### 咩係「項目」？

OpenCode 叫你揀一個資料夾，係因為佢需要一個**地盤**去做嘢：

```
你嘅項目資料夾/
├── 你擺入去嘅檔案（Excel、PDF、Word）
├── AI 幫你整好嘅結果
└── AI 用嚟做嘢嘅工具（自動生成，唔使你理）
```

簡單講：**你放啲原材料入去，AI 喺入面開工，做好嘅成品都放返喺入面。**

### 咩係「會話」？

每次你同 OpenCode 講嘢，就係開咗一個**會話**——好似 WhatsApp 入面一個獨立嘅 chat：

- 每個會話嘅內容係分開嘅，唔會撈亂
- 做完一個任務可以 close 咗佢，下次有新任務再開新嘅
- 如果 AI 理解錯咗，開個新會話由頭講過，仲快過同佢糾纏

> 一個任務用一個會話，做完就閂。簡單清楚。

### 操作

1. 新建一個資料夾（例如叫「Excel合併工具」）
2. 喺 OpenCode 中選擇呢個資料夾
3. 進入對話界面

## 🧠 4. 選擇 AI 模型

進入對話界面後，可以見到可用嘅模型列表。

OpenCode 默認提供**免費模型**，無需任何設定即可使用：

| 模型 | 說明 |
|---|---|
| **MIMO 2.5** | 小米出品，免費 |
| **DeepSeek V4 Flash Free** | DeepSeek 出品，免費 |

免費模型每日有大約 200 次調用額度，嘗鮮同學習完全夠用。

> 如果需要更穩定嘅服務，可以接入自己嘅 API Key（DeepSeek、GLM 等），後續課程會介紹。

## ⚠️ 安裝同常見問題

### Windows 安裝提示「無法打開」

右鍵安裝包 → 內容 → 剔選「解除鎖定」（如果有嘅話）→ 重新double click安裝。

<div className="h-4"></div>

### Mac 提示「無法驗證開發者」

1. 系統設定 → 私隱與安全性
2. 按「仍要打開」
3. 或者喺Terminal執行（直接 copy 下面指令貼上就得）：

```bash
xattr -cr /Applications/OpenCode.app
```

<div className="h-4"></div>

### 想用Terminal 版本

如果之後想用Terminal 版（功能更強大），可以喺Terminal執行：

```bash
# Mac
brew install opencode

# 或者用 npm
npm install -g opencode
```

## ❓ 常見問題

**Q：免費額度用完了怎麼辦？**

A：可以第二日再用（額度每日重置），或者接入自己嘅 API Key，按量付費。

**Q：桌面版打開好慢 / 卡住怎麼辦？**

A：桌面版第一次打開特別慢，係正常嘅。可以先用Terminal 版啟動一次升級版本，之後桌面版就會正常。
