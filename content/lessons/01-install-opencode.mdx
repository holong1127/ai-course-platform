## 2. 下載同安裝

### Step 1：打開官網

喺瀏覽器搜尋 **OpenCode**，入官網後按 **Download**。

[opencode 官網](https://opencode.ai/)

下載頁面提供三個版本：

| 版本 | 說明 |
|---|---|
| **桌面版（Desktop）** | 推薦新手使用，圖形界面，雙擊安裝 |
| **終端版（Terminal）** | 喺終端中運行，適合進階用戶 |
| **擴展版** | VS Code 等編輯器插件 |

### Step 2：下載安裝包

根據你嘅系統選擇：
- **Windows**：下載 `.exe` 安裝包
- **Mac**：下載 `.dmg` 安裝包

### Step 3：安裝

雙擊安裝包，按提示完成安裝。

> 安裝過程中唔需要任何配置，一路下一步就得。

## 3. 打開項目資料夾

安裝完成後，打開 OpenCode，會見到一個選擇項目資料夾嘅界面。

### 咩係「項目」？

可以理解為一個**工作空間**——就好似你做會計項目時嘅項目資料夾：

```
你選擇嘅資料夾/
├── 你放進去嘅數據文件（Excel、PDF 等）
├── AI 生成嘅工具代碼
├── AI 生成嘅結果文件
└── ...
```

- **你需要嘅數據**可以放進去
- **AI 生成嘅產物**都會喺裡面
- 做好**權限隔離**——AI 只能操作呢個資料夾，讀其他地方會問你

### 咩係「會話」？

每次你打開 OpenCode 開始一段對話，就係一個**會話（Session）**。

你可以理解為一個**聊天窗口**——好似微信裡同唔同人嘅聊天記錄：

- 每個會話係獨立嘅，互不干擾
- 閂咗 OpenCode 後，會話歷史會保留，下次打開仲可以繼續傾
- 喺同一項目中可以建立多個會話，例如一個做數據清洗、一個做報表分析
- 如果 AI 走偏咗，可以直接開一個新會話重新開始

> 實際使用中，一個大任務用一個會話就得。做完閂咗，下次有新任務再開新嘅。

### 操作

1. 新建一個資料夾（例如叫「Excel合併工具」）
2. 喺 OpenCode 中選擇呢個資料夾
3. 進入對話界面

## 4. 選擇 AI 模型

進入對話界面後，可以見到可用嘅模型列表。

OpenCode 默認提供**免費模型**，無需任何配置即可使用：

| 模型 | 說明 |
|---|---|
| **MIMO 2.5** | 小米出品，免費 |
| **DeepSeek V4 Flash Free** | DeepSeek 出品，免費 |

免費模型每日有大約 200 次調用額度，嘗鮮同學習完全夠用。

> 如果需要更穩定嘅服務，可以接入自己嘅 API Key（DeepSeek、GLM 等），後續課程會介紹。

## 5. 開始第一次對話

選擇模型後，就可以直接輸入文字同 AI 對話啦：

1. 喺輸入框中輸入你嘅需求（亦可以用語音輸入）
2. AI 會自動幫你完成

## ⚠️ 安裝同常見問題

### Windows 安裝提示「無法打開」

右鍵安裝包 → 內容 → 剔選「解除鎖定」（如果有嘅話）→ 重新雙擊安裝。

### Mac 提示「無法驗證開發者」

1. 系統設定 → 私隱與安全性
2. 按「仍要打開」
3. 或者喺終端執行：`xattr -cr /Applications/OpenCode.app`

### 想用終端版本

如果之後想用終端版（功能更強大），可以喺終端執行：

```bash
# Mac
brew install opencode

# 或者用 npm
npm install -g opencode
```

## 常見問題

**Q：免費額度用完了怎麼辦？**
A：可以第二日再用（額度每日重置），或者接入自己嘅 API Key，按量付費。

**Q：桌面版打開好慢 / 卡住怎麼辦？**
A：桌面版第一次打開特別慢，係正常嘅。可以先用終端版啟動一次升級版本，之後桌面版就會正常。
