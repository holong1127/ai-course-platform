## 本課要點

- 承接第 31 課，升級到有登入、有滑塊驗證碼嘅會員系統
- 登入用瀏覽器，查詢用 Code
- Cookie 重用機制

---

## 1. 真實痛點

注協嘅「企業雲」查詢權限免費、數據全（支援批量查 1 萬家、60 個維度），但企查查/天眼查接口好貴。

呢個 Skills 將免費權限用盡。

---

## 2. 核心解法

- **登入用瀏覽器**：滑塊驗證碼人手一次
- **查詢用 Code**：AI 捉 cookie 存文件，之後所有查詢純 Code 自動完成

---

## 3. 批量全維度匯出

上傳公司名單 Excel，自動勾選全部維度、輪詢下載狀態、解壓出每個維度一個 Excel。

---

## 4. Cookie 重用機制

登入一次憑證存文件，後續查詢帶 cookie + CSRF token 純 Code 請求。失效自動檢測、重新登入。

---

## 關鍵收穫

呢個方法可以套用到報稅系統、政府網站、公司內部系統——任何「要登入 + 重複操作」嘅場景。
