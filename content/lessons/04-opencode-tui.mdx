## 1. 點解要終端版？

桌面版雖然方便，但有以下限制：

- **資源佔用較高**——桌面版係 Electron 應用，比較食 RAM
- **更新較慢**——部分新功能終端版先有
- **自動化困難**——如果你想用 Script 呼叫 OpenCode，終端版先做到

終端版（TUI）係用鍵盤操作嘅文字界面，雖然睇落冇咁靚，但係：

- **更快更輕量**——唔使加載圖形界面
- **功能更完整**——支援更多進階操作
- **可以 SSH 遠程使用**——連去 Server 都用得

## 2. 安裝方法

### Mac 用戶

```bash
brew install opencode
```

### 或者用 npm（Windows / Mac 都得）

```bash
npm install -g opencode
```

國內用戶建議用鏡像：

```bash
npm install -g opencode --registry https://registry.npmmirror.com
```

### Windows 用戶注意

Windows 建議用 **PowerShell** 執行安裝指令。如果未裝 Node.js，先去 [nodejs.org](https://nodejs.org/) 下載安裝。

---

## 3. 基本操作

安裝完成後，喺終端輸入：

```bash
opencode
```

就會進入 TUI 界面。

### 常用快捷鍵

| 按鍵 | 功能 |
|---|---|
| `Shift+Tab` | 切換模式（Plan / Build / Orchestrator） |
| `Ctrl+C` | 中斷當前操作 |
| `Esc` (兩次) | 停止 AI 生成 |
| `/clear` | 清空當前會話 |
| `/compact` | 壓縮上下文，保留摘要 |
| `/exit` | 退出 OpenCode |
| `/goal` | 設定長期任務目標 |

### 模式切換

TUI 嘅模式切換同桌面版一樣：

- **Plan**：AI 先規劃，你確認再執行
- **Build**：AI 直接動手
- **Orchestrator**：多智能體協作模式（需安裝插件）

## 4. 終端版 vs 桌面版

| 對比 | 桌面版 | 終端版 |
|---|---|---|
| 安裝 | 雙擊安裝 | 一行指令 |
| 界面 | 圖形化 | 文字界面 |
| 速度 | 較慢（Electron） | 快（原生终端） |
| 資源佔用 | 高 | 低 |
| SSH 遠程 | 唔支援 | 支援 |
| 新手友好 | ✅ 推薦 | 需要適應 |

> 建議：新手先用桌面版，熟悉後可以試吓終端版。

## 常見問題

**Q：桌面版同終端版可以一齊用嗎？**
A：可以，佢哋共享同一套配置，互相唔影響。

**Q：終端版睇落好複雜，會唔會好難用？**
A：核心操作同桌面版一樣，只係輸入方式變咗做鍵盤。用慣咗就會發現仲快過用滑鼠。
