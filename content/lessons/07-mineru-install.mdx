## 1. MinerU 係咩？

MinerU 係一個開源嘅 PDF 解析引擎，GitHub 上有 66K+ Stars。

佢嘅核心能力係：**將任何 PDF（包括掃描件）轉換為 AI 可以讀懂嘅 Markdown 文字**。

### 點解需要 MinerU？

AI 本身唔可以直接讀 PDF——尤其係掃描件、有表格嘅文件。MinerU 就係 AI 嘅「眼睛」，幫佢睇明 PDF 入面嘅內容。

### 應用場景（會計財務）

- **提取發票資料**：自動辨識發票號碼、金額、日期
- **讀取銀行對帳單**：將 PDF 對帳單轉為結構化數據
- **合同審閱**：提取合同條款、甲方乙方、金額
- **年報分析**：讀取上市公司的 PDF 年報

## 3. 兩種模式

安裝完成後，MinerU 有兩種使用模式：

| 模式 | 特點 | 適合場景 |
|---|---|---|
| **Flash 模式** | 免登入、免配置、即用 | 簡單 PDF，唔需要高精度 |
| **精準模式** | 需 API Token，精度更高 | 複雜排版、表格多嘅 PDF |

### Flash 模式（推薦入門）

直接同 AI 講：

> 用 MinerU Flash 模式提取呢份 PDF 嘅內容

### 精準模式

如果你有 MinerU API Token：

> 用 MinerU 精準模式提取呢份合約嘅關鍵條款

## 常見問題

**Q：MinerU 準確嗎？**
A：對於印刷體 PDF，準確率超過 95%。手寫體會差啲。

**Q：要收費嗎？**
A：Flash 模式完全免費。精準模式需要 API Token，按量付費。
