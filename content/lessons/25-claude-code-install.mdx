## 2. 安裝四步

### Step 1：裝 Node.js（第4課已裝）

### Step 2：裝 Claude Code

```bash
npm install -g @anthropic-ai/claude-code
```

### Step 3：裝 cc-switch（接入國內模型）

```bash
npx cc-switch install
```

### Step 4：設定模型

填 API Key、模型檔位映射。
